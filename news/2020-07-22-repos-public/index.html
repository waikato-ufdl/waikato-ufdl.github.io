    Github repositories now publicly availably

    University of Waikato

    2020-07-22 09:26

    The (very much work-in-progress) code of the following UFDL repositories is now publicly available:

    • ufdl-backend - the Djano REST API backend

    • ufdl-json-messages - the JSON messages used by backend and Python client

    • ufdl-python-client - the core Python client encapsulating the REST API

    • ufdl-java-client - the core Java client encapsulating the REST API

    • ufdl-frontend-adams - ADAMS modules that make use of the Java client, providing access to the REST API within the workflow environment (include flows for managing backend and datasets)
